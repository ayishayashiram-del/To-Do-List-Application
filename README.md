# 📝 To-Do List Application

## 📌 Project Overview

This is a **menu-driven To-Do List Application** developed using **Python** as part of my **Python Programming **.

The application helps users organize and manage their daily tasks efficiently through a simple command-line interface.

## ✨ Features

- ➕ Add New Task
- 📋 View All Tasks
- ✏️ Update/Edit Task
- ✅ Mark Task as Completed
- ❌ Remove Task
- 🔍 Search Task
- 📊 Task Summary
- 🧹 Clear All Tasks
- 🚪 Exit Application
- ⚠️ Input Validation & Error Handling

## 🛠️ Technologies Used

- Python 3
- Lists
- Dictionaries
- Loops
- Conditional Statements
- Functions
- Exception Handling

## ▶️ How to Run

1. Install Python 3.
2. Download the project.
3. Open Terminal or Command Prompt.
4. Run the following command:

```bash
python todo.py
```
